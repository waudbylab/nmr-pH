NMR Buffer pH Estimation - Documentation

This application estimates sample pH from NMR buffer chemical shifts using nonlinear least-squares fitting. It supports multiple buffers, nuclei, and can refine temperature, ionic strength, and reference offsets.

Theory and Methods

pH-Dependent Chemical Shifts

The observed chemical shift of a buffer resonance depends on the population-weighted average of the limiting chemical shifts for each ionisation state:

δobs = Σ fi × δi (1)

where fi is the fractional population of ionisation state i, and δi is the limiting chemical shift for that state.

For a buffer with a single pKa, the fractional populations are given by the Henderson-Hasselbalch equation:

fprotonated = 1 / (1 + 10pH - pKa) (2)
fdeprotonated = 1 / (1 + 10pKa - pH) (3)

Temperature Dependence

The pKa varies with temperature according to the van't Hoff equation, with an optional heat capacity correction:

pKa(T) = pKa(Tref) + (ΔH/R)(1/T - 1/Tref) + (ΔCp/R)(Tref/T - 1 + ln(T/Tref)) (4)

where R = 8.314 J/(mol·K) is the gas constant, ΔH is the enthalpy of ionization, and ΔCp is the heat capacity change.

Chemical shifts also have temperature dependence, described by linear temperature coefficients:

δ(T) = δ(Tref) + αT × (T - Tref) (5)

where αT is the temperature coefficient in ppm/K.

Ionic Strength Dependence

The pKa is corrected for ionic strength using the Davies equation:

pKa(I) = pKa(I=0) + A·Δz2·(√I/(1+√I) - 0.3I) (6)

where A ≈ 0.51 M-1/2 at 25°C, Δz2 is the change in squared charge upon deprotonation, and I is the ionic strength in molarity.

The ionic strength is calculated from the molar concentrations and charges of all ions in solution:

I = 0.5 × Σ(ci × zi2) (7)

Chemical shifts may also have ionic strength dependence, described by linear coefficients:

δ(I) = δ(I=0) + αI × I (8)

where αI is the ionic strength coefficient in ppm/M.

Fitting Method

The application uses nonlinear least-squares fitting (Levenberg-Marquardt algorithm) to simultaneously determine pH from all observed chemical shifts. The fitting can optionally refine temperature, ionic strength, and/or chemical shift reference offsets.

Peak assignments are made automatically by matching observed shifts to the closest predicted resonance. Uncertainties in fitted parameters are estimated from the Jacobian matrix at the optimal solution.

Database Schema

The buffer database is a JSON file containing two main arrays: samples (experimental metadata) and buffers (buffer parameters).

Top-Level Structure

{
  "database_version": "0.0.1",
  "last_updated": "2025-01-01",
  "samples": [...],
  "buffers": [...]
}

Sample Object

Each sample describes the experimental conditions under which buffer data was measured:

FieldTypeDescription
sample_idstringUnique identifier (e.g., "waudby_2024_comprehensive_10pctD2O")
solventenumOne of: "10pct_D2O", "100pct_D2O", "H2O", "other"
authorsarrayList of {name, orcid, affiliation}
reference_temperature_KnumberReference temperature in Kelvin (typically 298.15)
reference_ionic_strength_MnumberReference ionic strength in M (typically 0.0)
measurement_rangesobjectValid ranges for pH, temperature_K, ionic_strength_M
temperature_calibrationstringDescription of temperature calibration method
pH_calibrationstringDescription of pH calibration method

Buffer Object

Each buffer contains thermodynamic and NMR parameters:

FieldTypeDescription
buffer_idstringUnique identifier (e.g., "waudby_2024_imidazole_10pctD2O")
buffer_namestringHuman-readable name
buffer_familystringBuffer family for grouping
sample_idstringReference to parent sample
ionisation_statesintegerNumber of ionisation states (pKa count + 1)
pKa_parametersarrayArray of pKa parameter objects
chemical_shiftsobjectKeyed by nucleus (1H, 13C, 15N, 19F, 31P)

pKa Parameters

Each pKa parameter object describes one deprotonation equilibrium:

FieldTypeDescription
pKa_indexinteger1-indexed pKa number
pKa[value, uncertainty]pKa at reference conditions
dH_kJ_mol[value, uncertainty]Enthalpy of ionisation
dCp_J_mol_K[value, uncertainty]Heat capacity change (optional)
protonated_chargeintegerCharge of the protonated form
ion_size_angstromnumberIon size for Debye-Hückel (optional)
Note: Values with uncertainties are represented as 2-element arrays: [value, uncertainty]. Single numbers can be used when uncertainty is not available.

Chemical Shifts

Chemical shifts are organized by nucleus, with each resonance having limiting shifts for each ionisation state:

"chemical_shifts": {
  "1H": [
    {
      "resonance_id": "H2",
      "description": "Imidazole H2 proton",
      "multiplicity": "singlet",
      "limiting_shifts": [
        {
          "ionisation_state": 0,
          "shift_ppm": [8.52, 0.01],
          "temperature_coefficient_ppm_per_K": [-0.0045, 0.0003],
          "ionic_strength_coefficient_ppm_per_M": [0.02, 0.01]
        },
        {
          "ionisation_state": 1,
          "shift_ppm": [7.68, 0.01],
          "temperature_coefficient_ppm_per_K": [-0.0038, 0.0003],
          "ionic_strength_coefficient_ppm_per_M": [0.03, 0.01]
        }
      ]
    }
  ]
}
Temperature and Ionic Strength Coefficients: The temperature_coefficient_ppm_per_K and ionic_strength_coefficient_ppm_per_M fields are optional. They describe how the limiting chemical shift changes with temperature or ionic strength for each ionisation state. When provided, these are used to predict chemical shifts at conditions different from the reference conditions.

Adding a New Buffer - Worked Example

Let's add imidazole as a new buffer. Imidazole has one pKa (~7.0) and two ionisation states.

Step 1: Check if sample exists

First, check if an appropriate sample entry exists for your solvent conditions. If not, create one: 

{
  "sample_id": "smith_2025_buffers_10pctD2O",
  "solvent": "10pct_D2O",
  "authors": [
    {
      "name": "Jane Smith",
      "orcid": "0000-0001-2345-6789",
      "affiliation": "University of Example"
    }
  ],
  "date_measured": "2025-01-15",
  "reference_temperature_K": 298.15,
  "reference_ionic_strength_M": 0.0,
  "measurement_ranges": {
    "pH": {"min": 4.0, "max": 10.0},
    "temperature_K": {"min": 278, "max": 318},
    "ionic_strength_M": {"min": 0.0, "max": 0.5}
  },
  "temperature_calibration": "Ethylene glycol chemical shift thermometer",
  "pH_calibration": "Glass electrode with NIST-traceable buffers"
}

Step 3: Validate your entry

Important checks:
  • Ensure ionisation_states = number of pKa values + 1
  • Each resonance must have limiting shifts for states 0 to (ionisation_states - 1)
  • Values with uncertainties must be 2-element arrays [value, uncertainty]
  • The sample_id must reference an existing sample
  • The buffer_id must be unique
